# Simpler

Simpler is simpler simplenote.com app for MacOS.

Features
- sidebar toggle
- font change
- color theme

## Download

https://github.com/swdyh/simplenote-electro/releases/latest

## Build

requirements
- electron 1.6.6
- electron-packager
- dotenv

```
make build
```
